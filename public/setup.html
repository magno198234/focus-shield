⚠️ Faça os dois passos abaixo para proteção completa no PC.

— Passo A: Extensão no Chrome —

1

Baixe a extensão

Clique no botão abaixo para baixar a pasta da extensão Focus Shield:

⬇️ Baixar Extensão

Clique em "Baixar tudo" no Google Drive para salvar a pasta no seu PC.

2

Abra o Chrome

Na barra de endereço do Chrome, digite:

chrome://extensions

Ative o Modo do desenvolvedor (canto superior direito).

3

Carregue a extensão

Clique em Carregar sem compactação e selecione a pasta da extensão que você baixou.
Se baixou como .zip, extraia a pasta primeiro.

— Passo B: NextDNS no Windows —

1

Abra as Configurações

Clique no menu Iniciar → Configurações → Rede e Internet → Wi-Fi.
Clique na sua rede Wi-Fi → Propriedades.

2

Edite o DNS

Role até Atribuição do servidor DNS → clique em Editar.
Mude para Manual e ative o IPv4.

3

Digite os endereços

DNS preferencial:

45.90.28.215

DNS alternativo:

45.90.30.215

Clique em Salvar.

✓

Pronto!

Todos os navegadores (Chrome, Edge, Firefox) estão protegidos.

Seguro: Apenas muda o servidor DNS. Não instala nada no PC.
