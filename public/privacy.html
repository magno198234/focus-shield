Política de Privacidade

Focus Shield — Última atualização: Abril de 2026

1. Introdução

A Focus Shield ("nós", "nosso") desenvolveu esta extensão de proteção digital com o objetivo de ajudar usuários a manterem um ambiente digital saudável. Esta Política de Privacidade descreve como tratamos as informações dos usuários da extensão Focus Shield para o Google Chrome.

2. Dados que NÃO coletamos

A extensão Focus Shield NÃO coleta, armazena, transmite ou compartilha nenhum dado pessoal do usuário, incluindo:

  • Histórico de navegação
  • URLs visitadas
  • Informações de identificação pessoal
  • Senhas ou credenciais
  • Localização geográfica
  • Dados de uso ou comportamento

3. Como a extensão funciona

A extensão Focus Shield opera inteiramente no dispositivo do usuário (localmente), sem enviar dados para servidores externos. Ela:

  • Monitora a navegação para detectar e bloquear URLs com conteúdo adulto
  • Verifica palavras-chave nas páginas para identificar conteúdo impróprio
  • Exibe uma página de bloqueio quando conteúdo inadequado é detectado
  • Todo o processamento ocorre localmente no navegador do usuário

4. Permissões utilizadas

A extensão solicita as seguintes permissões, todas necessárias para seu funcionamento:

  • webNavigation — para monitorar e interceptar navegação para sites com conteúdo adulto antes que o conteúdo seja carregado
  • host_permissions (<all_urls>) — necessário para analisar e bloquear conteúdo adulto em todos os sites visitados pelo usuário

5. Compartilhamento de dados

Não vendemos, trocamos, transferimos ou divulgamos de qualquer forma informações pessoais identificáveis para terceiros. A extensão não possui integração com serviços de análise, publicidade ou rastreamento.

6. Segurança

Como nenhum dado é coletado ou transmitido, não há riscos de violação de dados pessoais relacionados ao uso desta extensão.

7. Crianças

A extensão Focus Shield foi desenvolvida para proteger usuários de todas as idades, incluindo menores, de conteúdo adulto. Não coletamos intencionalmente informações de crianças menores de 13 anos.

8. Alterações nesta política

Podemos atualizar esta Política de Privacidade periodicamente. Notificaremos sobre quaisquer alterações publicando a nova política nesta página. Recomendamos que você revise esta página regularmente.
